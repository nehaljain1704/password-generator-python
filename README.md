 # Password Generator

A simple Python program that generates secure random passwords based on the user's desired length.

## Features

* Generate passwords of any length
* Uses uppercase letters, lowercase letters, numbers, and special characters
* Input validation for invalid password lengths
* Simple and beginner-friendly code

## Technologies Used

* Python 3
* random module
* string module

## How to Run

1. Clone the repository:

   ```bash
   git clone <repository-url>
   ```

2. Navigate to the project folder:

   ```bash
   cd PasswordGenerator
   ```

3. Run the program:

   ```bash
   python main.py
   ```

## Example

```text
Enter the length of password you need to generate: 10
Generated password: Ab7@K2m#P9
```
